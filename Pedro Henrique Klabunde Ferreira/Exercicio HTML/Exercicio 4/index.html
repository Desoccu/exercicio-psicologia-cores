O que é HTML Semântico?

HTML semântico é a utilização de tags que descrevem claramente o conteúdo de uma página, tornando-a mais compreensível tanto para humanos quanto para máquinas.

Benefícios da Semântica no HTML

Melhora a acessibilidade, facilita o SEO e organiza o código de forma mais estruturada.
